🔥 Firebase konfiguracija
Nije povezano
📋 Upute za postavljanje Firebase:
1. Idi na console.firebase.google.com
2. Projekt zelenilo-rovinj je već kreiran ✅
3. Build → Storage → Get started → Start in test mode → Done
4. Storage → Rules → zamijeni s: allow read, write: if true; → Publish
5. Unesi Firebase config ispod i klikni Poveži i sinkroniziraj

🔄 Real-time log
ℹ️ Kako radi sinkronizacija
• Svaka promjena se automatski sprema u Firebase oblak
• Svi uređaji vide promjene u realnom vremenu
• Radi offline — promjene se sinkroniziraju čim se veza uspostavi
• Podaci su sigurno pohranjeni na Google serverima
• Tekstualni podaci: besplatno do 1 GB (Realtime DB)
• Slike i dokumenti: besplatno do 5 GB (Firebase Storage)
• Slike se automatski uploadaju i vide na svim uređajima 📸
